文字列操作 in Rust

公開日：2020-05-26
更新日：-

Rust

文字アクセス

C++ みたいに添え字で文字アクセスすることはできない。

// C++でできること
Stirng a = "hoge_fuga";
a[0] // == 'h'

Rustではこうする。

let a = String::from("hoge_fuga");
a.char_inices().nth(0).unwrap().1 // == 'h'

逆順にアクセスしたいときはnth_back()を使う。
