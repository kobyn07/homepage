WindowsにおけるVS Codeでのショートカット集。 一部自分の使いやすいようにデフォルトから変更している点がある。

ショートカットの確認方法

左下の歯車アイコンから、キーボードショートカットをクリックする。 または、ctrl + Kのあとにctrl + Sを打つと開く（ショートカットを確認するショートカット）。

移動

内容 ショートカット
上下スクロール ctrl + ↑/↓
ファイルの先頭に移動 ctrl + home
ファイルの末尾に移動 ctrl + end
対応する括弧に移動 ctrl + shift + \

パネル操作

内容 ショートカット
パネル開閉 ctrl + i

Markdown編集

内容 ショートカット
プレビュー ctrl + shift + V
$$挿入 ctrl + M
目次作成 ctrl + T + ctrl + T
