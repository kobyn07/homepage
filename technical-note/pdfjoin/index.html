PDFの結合 (pdfjoin)

公開日：2020-06-24
更新日：-

pdfcommandline

pdfjoinとは

pdfの結合ツール。 pdfjamというパッケージの一部らしい。

使い方

hoge.pdf, fuga.pdf を結合したいとき、

pdfjoin hoge.pdf fuga.pdf -o output.pdf

連番ファイルの場合楽ができて、 hoge1.pdf, ..., hoge6.pdf とすると、

pdfjoin hoge[1-6].pdf -o output.pdf

変な回転を止めさせる

上のコマンドを使うと勝手に回転されたpdfが出てくるときがある。 単純に結合したいときは、--rotateoversize falseのオプションを付け加える。
