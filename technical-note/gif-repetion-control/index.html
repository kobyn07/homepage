gif の繰り返し制御

公開日：2020-06-26
更新日：2020-06-27

gifcommandline

インストール

sudo apt install gifsicle

使い方

無限ループ

gifsicle -l input.gif > output.gif

-l=n で n 回繰り返すようにできる。

繰り返さない

gifsicle --no-loopcount input.gif > output.gif
