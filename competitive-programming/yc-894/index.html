yukicoder No.894 二種類のバス

公開日：2019-09-28
更新日：2019-09-28

yukicoder数学

問題

方針

T-1までの時間に、0と、AまたはBの倍数の時間が何個あるか数えたい。 よってT-1をAとBで割った数から、T-1をAとBの最小公倍数で割った数を引いた値が答えとなる。

気をつけたいのが制約でAとBが最大\(10^{18}\)なので、最小公倍数がunsigned long longの範囲を超える。 C++17の機能にはlcm関数を使えるところだが、オーバーフローするので注意すること。

C++で多倍長整数はめんどくさいので、素直に Python とかに切り替えたほうがよい。

C++17(GCC)のunsigned long longで計算すると、オーバーフローするけど、Tより大きな数なのでACしちゃった…

解答

import math

def lcm(x, y):
    return (x*y)//math.gcd(x, y)

T, A, B = map(int, input().split())
T -= 1
ans = T//A + T//B + 1
ans -= T//lcm(A, B)
print(int(ans))
