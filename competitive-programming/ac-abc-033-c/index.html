AtCoder Beginner Contest 033 C - 数式の書き換え

公開日：2019-11-21
更新日：-

AtCoder場合分け注意力

問題

方針

数式\(S\)には演算子が+と*しか出てこない。 演算子は*のほうが高く、それ以上の計算順序は存在しない。 目標は式の計算値を0にしたい。 つまり、+で区切られた区間が0になればよい。

解法

#include <bits/stdc++.h>
using namespace std;

int main() {
    string s;
    cin >> s;

    int ans = 0;
    int cnt = s[0] - '0';

    for (size_t i = 1; i < s.size(); i += 2) {
        if (s[i] == '+') {
            if (cnt)
                ans++;
            cnt = s[i + 1] - '0';
        } else {
            cnt *= s[i + 1] - '0';
            // オーバーフローの可能性がある
            if (cnt)
                cnt = 1;
        }
    }
    if (cnt)
        ans++;
    cout << ans << endl;

    return 0;
}
