AtCoder Regular Contest 004 A - 2点間距離の最大値 ( The longest distance )

公開日：2020-01-19
更新日：-

AtCoder全探索

問題

方針

素直に全探索で間に合う。

計算量：\(O(N^2)\)

AC解

便利な関数 std::hypot

2点間の距離を計算するのにいちいち \(\sqrt{(x_i - x_j)^2 + (y_i - y_j)^2}\) とかめんどくさい。 そこで便利な関数 std::hypot を使う。 上の式と同等の計算をしてくれるので便利。

double dist = std::hypot((xi - xj), (yi - yj));
