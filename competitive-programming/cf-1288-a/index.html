Codeforces 1288 A. Deadline

公開日：2020-01-16
更新日：-

CodeforceswolframAlpha

問題

方針

\(d\)日かかるプログラムがある。 \(n\)日までにプログラムを完了する必要がある。 開始から、\(x\)日目にプログラムを最適化したので、\(x + \lceil \frac{d}{x+1}\rceil\)日でプログラムを終了することができた。 \(n\)日までに終了するかYES/NOで答えよ。

プログラム最適化後の\(x + \lceil \frac{d}{x+1}\rceil\)の動きがよくわからないので、WolframaAlpha 先生 に聞いてみる（リンク先は適当に \(d=10\)にした。） ほしいのは \(x\)の最小値が \(n\) 以下かどうかである。 極小値の欄をみると \(x = \sqrt{d}-1\)が答えらしいので、その付近を適当にシミュレーションする。

解法

(https://codeforces.com/contest/1288/submission/68916680)

#pragma region Template

int main() {
    cin.tie(0);
    ios::sync_with_stdio(false);

    int T;
    scan(T);
    rep(_, T) {
        int n, d;
        scan(n, d);
        bool f = false;
        rep(x, sqrt(d) + 2) {
            int day = x + ceil(d / (x + 1.0));
            if (!f && day <= n) {
                debug(day);
                f = true;
            }
        }
        if (f)
            print("YES");
        else
            print("NO");
    }

    return 0;
}
