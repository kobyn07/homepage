AtCoder Beginner Contest 046 C - AtCoDeerくんと選挙速報

公開日：2019-10-02
更新日：-

AtCoderABCシミュレーション数学

問題

方針

\(T_i : A_i\)の比が成り立つ最小の数の和を求めたい。

制約の比は互いに素であることから、\(i = 1\)のときの比が最初の得票数であることがわかる。

\(i<=2\)のとき、それまでの最小の数をそれぞれ\(t, a\)とする。 \(t, a\) から \(T_i : A_i\)の比になるような最小の値を探したい。 \[ \begin{eqnarray} \left\{ \begin{array}{l} t \leq nT_i \\ a \leq nA_i \end{array} \right. \end{eqnarray} \] 上の式から条件に合った最小の\(n\)を探せばよいことがわかる。 \(n = max(T/t, A/a)\)（割り算はすべて切り上げ）が最適な\(n\)である。

解答

#include <bits/stdc++.h>
using namespace std;

template <class T> T up(T a, T div) { return (a + div - 1) / div; }

int main() {
    int N;
    cin >> N;
    vector<long long> T(N), A(N);
    for (int i = 0; i < N; i++) {
        cin >> T[i] >> A[i];
    }
    long long a = A[0], t = T[0];
    for (int i = 1; i < N; i++) {
        long long n = max(up(a, A[i]), up(t, T[i]));
        t     = n * T[i];
        a     = n * A[i];
    }
    cout << t + a << endl;

    return 0;
}
