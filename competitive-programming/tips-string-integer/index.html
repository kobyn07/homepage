string型（char型）と数値の相互変換

公開日：2019-10-05
更新日：-

tips

char -> string

一様初期化でstring型にする。

char c = 'c'
string s = string{c};

string -> char

stringは要素アクセスでchar型に変換される。

string s = "string";
char c = s[0]; // c=='c'

char -> int

char型の'0'と相対的な差を取ることでint型に変換できる。

char c = '9';
int i = c - '0'; // i==9

string -> int（数値型）

便利な関数std::stoi()があるのでそれを使う。

string s = "100";
int i = std::stoi(s); // i==100

long long型はstd::stoll()を使う。 他の数値型はcpprefjp - C++日本語リファレンス std::stoiの下の方にある関連リンク -> ファミリーあたりを参照すること。

int（数値型）-> string

便利な関数std::to_string()を使う。

int i = 100;
string s = std::to_string(i);
